Reporting Bugs

IP Traffic Spy has been tested on Windows 98, ME, NT4, 2000 & XP.

The windows 95 kernel does not have sufficient threading support to run IP Traffic Spy, therefore IP Traffic Spy will not run on a windows 95 platform.

In the past releases of IP Traffic Spy few bugs have been reported but all found have been fixed. If IP Traffic Spy crashes or gives you an error, please take note of any error messages and if possible also note the circumstances in which it happened. Then visit the bug reporting page on NetworkDLS's website and submit any collected error information.
