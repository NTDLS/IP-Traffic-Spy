Exclude Filters

    The Exclude Filters dialog allows you to setup filters that will be applied to encountered TCP/IP packets to decide whether they should be excluded in the packets you want to see.



Source IP Addresses allows you to enter source IPs to exclude.
Destination IP Addresses allows you to enter destination IPs to exclude.
Source Ports allows you to enter source ports to exclude.
Destination Ports allows you to enter destination ports to exclude.

Notes:
    If the received TCP/IP packet matches any of the defined "Exclude" characteristics, the packet will not be written to Data Grid Dialog even it also matches one or more of the Include Filter characteristics.
