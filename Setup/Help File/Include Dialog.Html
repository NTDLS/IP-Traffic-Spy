Include Filters

    The Include Filters dialog allows you to setup filters that will be applied to encountered TCP/IP packets to decide whether they should be included in the packets you want to see.



Source IP Addresses allows you to enter source IPs to include.
Destination IP Addresses allows you to enter destination IPs to include.
Source Ports allows you to enter source ports to include.
Destination Ports allows you to enter destination ports to include.

Notes:
    If the received TCP/IP packet matches any of the defined "Include" characteristics, the packet will be written to Data Grid Dialog unless it also matches one or more of the Exclude Filter characteristics.
